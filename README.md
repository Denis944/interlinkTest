# interlinkTest

## Description

1) В данному завданні я наводжу приклади використання принципів ООП на мові програмування Java.
  За приклад я взяв трьох бійців різних стилів:Боксер(Boxer), Кікбоксер(Kickboxer) та 
  боець ММА (MMAFighter). Всі три бійці є потомками
  абстрактного класу Fighter. До класу Боксеру було додано поле урон з руки.  Kickboxer  наслідується від класу Boxer та розширяє його 
  за технікою(додаються удари ногами та урон з ноги) також у цьому класі є константа LEG_DAMAGE_KOFFICIENT,яка приймає участь у разрахунку
 урону(це показує,що удари ногами є менш ефективними за удари руками).До класу MMAFighter було додане поле борцовський урон. Всі бійці реалізують 
інтерфейс FighterInterface у якому є два методи: удар та блок,але клас MMAFighter
також  реалізує інтерфейс WrestlerTechniques, який розширений борцовською технікою
(методи : підніжка, захват, кидок).
2) У методі Main() за допомогою спільного інтерфесу FighterIntrface була показана
 поліморфність вищезазаначених сутностей.
