# interlinkTest

## Description

1) В данному завданні я наводжу приклади використання принципів ООП на мові програмування Java.
  За приклад я взяв трьох бійців різних стилів:Боксер(Boxer), Кікбоксер(Kickboxer) та 
  ГрекоРимський Борець (GreekRomanWrestler). Всі три бійці є потомками
  абстрактного класу Fighter.  Kickboxer  наслідується від класу Boxer та розширяє його 
  за технікою(додаються удари ногами). Всі бійці реалізують інтерфейс FighterInterface
   у якому є два методи: удар та блок,але клас GreekRomanWrestler
також  реалізує інтерфейс WrestlerTechniques, який розширений борцовською технікою
(методи : підніжка, захват, кидок).
2) У методі Main() за допомогою спільного інтерфесу FighterIntrface була показана
 поліморфність вищеописаних сутностей.
